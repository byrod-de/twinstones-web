Features

Commands & Features

What it does: Rolls two d12 (Hope + Fear).
Options:
- modifier – adds/subtracts a number (default 0)
- bonus – advantage, disadvantage, ally (adds extra d6)
- ally_dice – number of d6 for ally help (only with bonus=ally)
- type – Action (default) or Reaction
- d20_bonus – hope, fear, or none
- show – set false to hide
Example: /dh-roll modifier:2 bonus:advantage type:Reaction

What it does: Rolls standard dice or coin.
Options:
- dice – XdY (e.g., 3d6) or coin (default 1d20)
- bonus – only for single d20: advantage/disadvantage
- modifier – adds/subtracts number
- show – set false to hide
Examples:
/dice-roll dice:coin
/dice-roll dice:3d6
/dice-roll dice:d20 bonus:advantage

What it does: Rolls any mix of dice and numbers.
Input: put everything in rollinput, e.g., 2d6 + d8 - 2
Allowed dice: d2, d4, d6, d8, d10, d12, d20, d100
show – set false to hide
Example: /roll-any rollinput:2d6 + d8 - 2
